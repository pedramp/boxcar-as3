Constructor Detail
Boxcar()Constructor
public function Boxcar(key:String, secret:String)

Boxcar ActionScript API is a simple way to using Boxcar notification service this class has simple rule and you can use it as simple as eating a frog! :D here we have just three method to sending notification to a user OR to many user or to subscribe user.

Parameters
key:String — Provider Key
 
secret:String — Provider Secret,

SecterKey is optional when you want send a notify to direct user but for broadcasting and subscription you should fil it.

See also

Example
however we have three simple classes that you can work with them directly., just for example if you want send a message to a user: 
var n:INotification = new NotifyToOne('my_provider_key', 'receiver_email_address', 'my message!');
n.message = 'Test API';
n.sourceURL = 'http://example.com';
n.send();
and you can handle all HTTP requestes by default URLLoader events

Example
you can send 
// setting my provider key
var box:Boxcar = new Boxcar('fKJ9a8x9Q8QXSwzADwhy');
box.notify('receiver_email_address', 'Hello!').send();
also you can config other optioal keys like fromScreenName, iconURL or... 
// setting my provider key
var box:Boxcar = new Boxcar('fKJ9a8x9Q8QXSwzADwhy');
var notify:INotification = box.notify('i[at]pedram.mp', 'Hello!');
notify.iconURL = 'http://example.com/57x57.png';
notify.fromScreenName = 'Sample!';
notify.send();
// adding event to getting http-status
notify.addEventListener(HTTPStatusEvent.HTTP_STATUS, onEvent);
function onEvent(e:HTTPStatusEvent):void
{
    trace(e.status);
}
subscribe()method 
public function subscribe(email:String):INotification

his sends the user with the matching e-mail address a push notification which requests that they add your service. This will ease integration with your website, as you will have already registered them on your system and can identify whom they are based on their Boxcar e-mail address.

Parameters

email:String — The Boxcar users e-mail address

Returns
INotification — INotification is your request instance
